## 📝 手动测试流程

### P1-1: 周/日视图测试

**前置条件**:
- 应用已启动
- 已创建至少 5 个任务

**测试步骤**:

1. **视图切换测试**
   ```
   1. 点击顶部工具栏 "Week" 按钮
   2. 观察视图切换动画
   3. 点击 "Day" 按钮
   4. 观察日视图显示
   5. 点击 "Month" 按钮返回月视图
   ```
   
   **预期**: 视图切换流畅 (<500ms)，布局正确

2. **周视图布局验证**
   ```
   1. 切换到周视图
   2. 检查左侧时间轴 (08:00-20:00)
   3. 检查顶部日期 (周一 - 周日)
   4. 检查网格线对齐
   ```
   
   **预期**: 布局正确，无错位

3. **日视图布局验证**
   ```
   1. 切换到日视图
   2. 检查 24 小时时间轴
   3. 检查右侧任务列表
   4. 检查时间刻度
   ```
   
   **预期**: 时间轴完整，任务列表清晰

4. **视图联动测试**
   ```
   1. 在月视图选择 3 月 15 日
   2. 切换到周视图
   3. 确认显示 3 月 15 日所在周
   4. 切换到日视图
   5. 确认显示 3 月 15 日
   ```
   
   **预期**: 日期选择保持同步

5. **任务显示测试**
   ```
   1. 在周视图查看任务分布
   2. 确认任务在正确日期列
   3. 检查优先级颜色 (红/黄/绿)
   4. 在日视图查看任务时间段
   ```
   
   **预期**: 任务位置准确，颜色正确

6. **视图导航测试**
   ```
   1. 在周视图点击 "上周" 按钮
   2. 确认显示上周
   3. 点击 "下周" 按钮
   4. 确认显示下周
   5. 在日视图测试 "前一天"/"后一天"
   ```
   
   **预期**: 导航正确，日期更新

**通过标准**:
- ✅ 所有步骤预期结果符合
- ✅ 视图切换时间 <500ms
- ✅ 无 UI 错位或渲染问题

### P1-2: 任务搜索测试

**前置条件**:
- 应用已启动
- 已创建至少 10 个任务

**测试步骤**:

1. **关键词搜索**
   ```
   1. 在搜索框输入 "项目"
   2. 观察搜索结果
   3. 检查匹配任务高亮
   4. 检查非匹配任务变淡
   ```
   
   **预期**: 实时显示结果，高亮清晰

2. **模糊搜索**
   ```
   1. 输入不完整关键词 "项"
   2. 观察搜索结果
   ```
   
   **预期**: 匹配所有包含"项"的任务

3. **搜索高亮**
   ```
   1. 搜索 "报告"
   2. 查看匹配任务
   3. 检查关键词高亮
   ```
   
   **预期**: 黄色背景高亮，醒目清晰

4. **搜索清除**
   ```
   1. 执行搜索
   2. 点击清除按钮 (×)
   3. 观察任务显示
   ```
   
   **预期**: 所有任务恢复显示，高亮消失

5. **无结果处理**
   ```
   1. 搜索 "xyz123abc"
   2. 观察结果
   ```
   
   **预期**: 显示"未找到匹配任务"

6. **大小写不敏感**
   ```
   1. 搜索 "PROJECT"(大写)
   2. 搜索 "project"(小写)
   3. 比较结果
   ```
   
   **预期**: 两次搜索结果相同

7. **搜索 + 过滤组合**
   ```
   1. 执行搜索 "任务"
   2. 应用 P0 优先级过滤
   3. 检查结果
   ```
   
   **预期**: 搜索和过滤同时生效

**通过标准**:
- ✅ 搜索响应时间 <300ms
- ✅ 搜索结果准确
- ✅ 高亮显示清晰

### P1-3: 任务过滤测试

**前置条件**:
- 已创建 10 个任务
- P0/P1/P2 各 3-4 个
- 待办/已完成各若干

**测试步骤**:

1. **优先级过滤**
   ```
   1. 点击过滤按钮
   2. 选择 "P0-高优先级"
   3. 检查显示结果
   4. 重复测试 P1、P2
   ```
   
   **预期**: 仅显示对应优先级任务

2. **状态过滤**
   ```
   1. 选择 "待办" 过滤
   2. 检查结果
   3. 选择 "已完成" 过滤
   4. 检查结果
   ```
   
   **预期**: 状态过滤准确

3. **组合过滤**
   ```
   1. 选择 "P0" 过滤
   2. 再选择 "待办" 过滤
   3. 检查结果
   4. 检查过滤标签
   ```
   
   **预期**: 显示 P0 且待办的任务

4. **清除过滤**
   ```
   1. 应用多个过滤
   2. 点击 "清除" 按钮
   3. 检查结果
   ```
   
   **预期**: 所有过滤移除，全部显示

5. **逾期过滤**
   ```
   1. 创建逾期任务
   2. 选择 "逾期" 过滤
   3. 检查结果
   ```
   
   **预期**: 仅显示逾期任务

6. **今日过滤**
   ```
   1. 选择 "今日" 过滤
   2. 检查结果
   ```
   
   **预期**: 仅显示今日任务

**通过标准**:
- ✅ 过滤准确无误
- ✅ 组合过滤正确
- ✅ 显示匹配数量
