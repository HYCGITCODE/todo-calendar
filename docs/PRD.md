## 🎯 项目目标

打造一个**开箱即用**的桌面待办任务管理工具，让用户通过直观的日历视图管理任务，支持双击创建、拖拽修改日期，实现高效的任务管理体验。

## 🏗️ 技术栈确认

### 核心技术栈
| 组件 | 技术选型 | 理由 |
|------|----------|------|
| **GUI 框架** | **Tkinter** (首选) / PyQt5 (备选) | Tkinter 内置无需额外安装，PyQt5 更美观但需打包 |
| **数据存储** | **SQLite** | 轻量级，单文件，无需配置，适合桌面应用 |
| **打包工具** | **PyInstaller** | 成熟稳定，支持 Windows 单文件打包 |
| **编程语言** | **Python 3.8+** | 快速开发，跨平台，生态丰富 |

### 技术决策说明
1. **为什么选 Tkinter 而不是 PyQt？**
   - Tkinter 是 Python 内置库，减少打包体积
   - 学习曲线低，开发速度快（12 小时挑战关键）
   - 功能足够实现核心需求
   - **备选**: 如时间充裕可升级到 PyQt5 提升 UI 美观度

2. **为什么选 SQLite？**
   - 单文件数据库，无需安装配置
   - 支持 SQL 查询，方便后续扩展
   - 性能足够桌面应用使用

3. **打包策略**
   - 使用 PyInstaller 生成单文件可执行程序
   - 包含 `--onefile --windowed` 参数
   - 目标：用户只需下载一个 `.exe` 文件
