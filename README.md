# AI News Pulse 🚀

AI 行业新闻聚合 Web 应用 - 5 小时极速开发 MVP

## 项目简介

AI News Pulse 是一个轻量级的 AI 行业新闻聚合应用，旨在为用户提供最新、最热门的 AI 行业资讯。通过聚合多个权威来源，帮助用户快速掌握 AI 领域动态。

## 核心特性

- 📰 实时聚合 AI 行业新闻
- 🔥 热门话题追踪
- 📱 响应式设计，支持移动端
- ⚡ 极速加载，简洁界面

## 技术栈

- **前端**: HTML5 + TailwindCSS + Vanilla JavaScript
- **后端**: Node.js + Express
- **数据源**: RSS  feeds + API 聚合

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```
