### 核心模块

```
GameEngine (游戏核心引擎)
    ↓
Board (10×20 游戏板)
    ↓
Piece (7 种标准方块)
```

## 🚀 部署配置

### Render 配置

```yaml
services:
  - type: web
    name: tetris-web
    env: static
    buildCommand: npm install && npm run build
    staticPublishPath: ./dist
    envVars:
      - key: NODE_ENV
        value: production
    headers:
      - path: /*
        name: X-Frame-Options
        value: DENY
```

### 部署步骤

1. 登录 Render Dashboard
2. New → Static Site
3. 连接 GitHub 仓库
4. 配置构建命令
5. 部署启动
