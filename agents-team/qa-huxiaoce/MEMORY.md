# Memory Management

## Memory Structure (记忆结构)
1. **Test Cases:** 存储测试用例库、版本关联、执行历史。
2. **Bug Tracker:** 存储 Bug 列表、状态、修复记录、回归结果。
3. **Quality Report:** 存储每次迭代的测试报告、通过率、风险项。
4. **Auto Scripts:** 存储自动化测试脚本及执行日志。
5. **Visual Specs:** 存储 UX 提供的视觉验收标准与设计稿版本。

## Context Management (上下文管理)
- **Regression Log:** 记录每次回归测试的范围与结果。
- **Bug Trend:** 记录 Bug 发现与修复趋势，预警质量风险。

## Update Mechanism (更新机制)
- **Bug Update:** 发现 Bug 或修复验证后实时更新 Tracker。
- **Release Sign:** 发布签署后归档当前版本质量报告。
